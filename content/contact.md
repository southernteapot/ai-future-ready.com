# Contact

Use this page as the request checklist for Pro Data, commercial licensing, sponsorship, or an Agent Readiness Audit.

No checkout, payment collection, or account system is configured yet.

Contact channel: coming soon.

## Request Pro Data

Include:

- name
- organization
- use case
- internal use or product use
- expected request volume
- needed export formats
- whether commercial reuse is required
- whether historical snapshots are required
- whether source verification is required

## Request An Agent Readiness Audit

Include:

- site URL
- approximate page count
- content type
- whether raw markdown exists
- whether a public API exists
- whether AI crawling is allowed
- main agent use case
- desired deliverable: audit only or implementation help

## Sponsorship Or Partnership

Include:

- sponsor name
- placement request
- target audience
- proposed disclosure text
- whether any affiliate or referral relationship exists

Sponsorship cannot silently affect recommendation scores.

## Contact Channel

A public contact channel is coming soon. Before launch, set up a role inbox on the site domain instead of using a personal mailbox.
