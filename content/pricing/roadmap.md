# Public Roadmap

This roadmap keeps the site honest about what exists now and what is still draft.

## Now

- Plain markdown-style interface.
- Raw markdown access.
- `llms.txt` and `llms-full.txt`.
- Public JSON API.
- Per-item JSON.
- Search index.
- Recommendation endpoints.
- Change feed.
- Provider verification metadata.
- Pro Data draft.
- Commercial license draft.
- Agent Readiness Audit draft.
- Public Pro Data sample.
- Pricing snapshot endpoint.

## Next

- Add source metadata to every model, not only providers.
- Add field-level price source records.
- Add stronger recommendation explanations.
- Add historical pricing snapshots.
- Add hash manifest.
- Add data changelog with typed change records.
- Add intake process for audits and data access.
- Add more provider profiles.
- Add more model-router guidance.

## Later

- Pro API authentication.
- Bulk NDJSON export.
- CSV exports.
- Webhooks for change alerts.
- Commercial license finalization.
- Team and enterprise support terms.
- Private verification requests.

## Not Planned Yet

- checkout
- account dashboard
- display ads
- hidden paid ranking influence
- JavaScript-heavy application shell
